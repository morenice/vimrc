# vimrc
vim essential configuration
* c/c++ development
* python development
* php development


# configure guide

Install ctags

`$ sudo apt-get install ctags`

Set up vim-plug

```sh
$ curl -fLo ~/.vim/autoload/plug.vim --create-dirs \
    https://raw.githubusercontent.com/junegunn/vim-plug/master/plug.vim
```

Clone vimrc

`$ cp vimrc ~/.vimrc`

`vim` and run `:PlugInstall`
