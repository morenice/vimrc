# vimrc
vim essential configuration
* c/c++ development
* python development
* web development


# configure guide
Install ctags, tmux

`$ sudo apt-get install ctags tmux`

Set up vim-plug

```sh
$ curl -fLo ~/.vim/autoload/plug.vim --create-dirs \
    https://raw.githubusercontent.com/junegunn/vim-plug/master/plug.vim
```

Clone vimrc

`$ cp vimrc ~/.vimrc`

`vim` and run `:PlugInstall`
