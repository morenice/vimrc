# vimrc
vim essential configuration
* c/c++ development
* python development
* web development


# configure guide
Install ctags, tmux

linux
```$ sudo apt-get install ctags tmux```

macOS
```$ brew install ctags tmux```

Copy vimrc

```$ cp vimrc ~/.vimrc```

`vim` and run `:PlugInstall`
